# Set 3: Array Algorithms & Keys Generation (Problems 21-30)

This folder focuses on **Array Manipulations**, **Randomness**, and **Data Generation**. It covers fundamental algorithms for processing lists of numbers and generating secure-like keys.

## Problem List

| # | Problem Name | Description | Key Concepts |
|---|---|---|---|
| 21 | **Generate Keys** | Generates random serial keys (e.g., XXXX-XXXX-XXXX-XXXX). | `Random Char`, `Loops`, `String Manipulation` |
| 22 | **Repeated Elements Count** | Counts how many times a specific number appears in an array. | `Arrays`, `Linear Search`, `Counter Logic` |
| 23 | **Fill Array With Random** | Fills an array with random numbers within a specific range. | `rand()`, `Arrays`, `MyLib Integration` |
| 24 | **Max Number In Array** | Finds the maximum number in a randomly generated array. | `Arrays`, `Comparison Logic`, `Optimization` |
| 25 | **Min Number In Array** | Finds the minimum number in a randomly generated array. | `Arrays`, `Comparison Logic` |
| 26 | **Sum Array Numbers** | Calculates the sum of all elements in an array. | `Accumulator Pattern`, `For Loops` |
| 27 | **Average Array Numbers** | Calculates the average value of array elements. | `Type Casting`, `Arithmetic Logic` |
| 28 | **Copy Array** | Copies elements from one array to another. | `Array Traversal`, `Index Management` |
| 29 | **Copy Prime Numbers** | Filters and copies only prime numbers to a new array. | `Conditional Logic`, `MyLib::Is_Prime` |
| 30 | **Sum Of 2 Arrays** | Sums elements of two arrays index-by-index into a third array. | `Parallel Arrays`, `Vector Logic` |

## Key Features
* **Array Algorithms:** mastered traversal, searching, filtering, and aggregation.
* **Random Data Generation:** Simulating real-world data scenarios using `rand()` and `MyLib`.
* **Key Generation:** Building complex strings from random components.
* **Reusability:** Heavily reliant on `MyLib` for random numbers and validations.
